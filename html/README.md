# 阵营射击大战

一个基于阵营的多人射击游戏，支持最多12名玩家进行阵营对抗。

## 文件结构

- `main.html` - 主界面，游戏启动入口
- `index.html` - 游戏设置界面
- `script.js` - 游戏逻辑和设置界面逻辑

## 游戏流程

1. **主界面** (`main.html`)：显示游戏标题和单机游戏按钮
2. **设置界面** (`index.html`)：配置玩家人数、阵营和控制键
3. **游戏进行**：在设置界面中进行游戏
4. **游戏结束**：显示获胜阵营和排行榜，可选择重新开始或回到主界面

## 特色功能

- 🎯 **阵营系统**：6个不同颜色的阵营
- 🏰 **领土占领**：通过战斗占领地图墙壁
- 📊 **详细统计**：击杀数和伤害统计排行榜
- 🎮 **灵活控制**：支持键盘和鼠标控制
- 📱 **触控支持**：移动设备友好

## 启动方式

直接在浏览器中打开 `main.html` 开始游戏体验！

## 游戏规则

- 玩家分为6个阵营（红、蓝、绿、黄、粉、青）
- 通过射击占领墙壁来扩大领土
- 击败所有敌对阵营的玩家即可获胜
- 同一阵营的玩家和子弹可以互相穿过队友占领的墙壁
